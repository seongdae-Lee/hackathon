## 1. 프로젝트 개요

### 목적

건강 목표 달성에 도움이 되는 게임을 AI 기반으로 큐레이션하여 제공하는 풀스택 웹 애플리케이션입니다. Claude API를 활용해 게임의 건강 효과를 자동 분류하고, 사용자에게 맞춤형 건강 게임을 추천합니다.

### 핵심 가치

- 게임을 통한 건강 목표 달성 동기 부여
- AI(Claude API) 기반 건강 효과 자동 태깅 및 분류
- 직관적인 UI/UX로 게임 탐색 및 추천 제공

## 8. 실행 방법

### 사전 요구사항

- .NET 9 SDK (또는 .NET 8+)
- Node.js 18+
- npm

### 백엔드 실행

```bash
cd backend
dotnet run --project src/HealthGameCurator.Api --urls "http://localhost:5000"
```

- 첫 실행 시 EF Core 마이그레이션 및 시드 데이터(21개 게임) 자동 적용
- SQLite DB 파일: `backend/src/HealthGameCurator.Api/healthgamecurator.db`
- Swagger UI: `http://localhost:5000/swagger`

### Claude API 키 설정 (AI 태깅 활성화)

```bash
# backend/src/HealthGameCurator.Api/appsettings.json에서 설정
# 또는 환경변수로
export ClaudeApi__ApiKey="your_api_key_here"
```

> API 키 없이도 카테고리 기반 Mock 태그로 동작합니다 (Graceful degradation).

### 프론트엔드 실행

```bash
cd frontend
npm install
npm run dev
```

- 접속 URL: `http://localhost:3000`

### Docker 전체 스택 실행

```bash
# .env 파일 생성
echo "CLAUDE_API_KEY=your_api_key_here" > .env

docker compose up --build
```
